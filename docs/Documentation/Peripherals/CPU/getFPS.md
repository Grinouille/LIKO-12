# CPU.getFPS
---

Returns the current frame per second count.

---

* **Available since:** _CPU:_ v1.0.0, _LIKO-12_: v0.6.0
* **Last updated in:** _CPU:_ v1.0.0, _LIKO-12_: v0.6.0

---
# Syntax
---

```lua
local fps = CPU.getFPS()
```

# Arguments
---



---

* **fps (number):** The FPS.
